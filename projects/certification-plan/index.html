Industry Certification Preparation Plan

Web & Game Design / JavaScript Programming

Instructor

Mr. Montevious Stevenson

Program Goal

Prepare students to successfully earn a JavaScript Programming Industry-Based Certification by developing technical knowledge, problem solving skills, and applied programming experience.

Phase 1: Foundations of Programming

Students begin by learning core programming concepts and understanding how JavaScript interacts with HTML and web applications.

  • Variables and data types
  • Conditional statements
  • Loops
  • Functions
  • Basic DOM manipulation

Phase 2: Applied Web Development

Students apply their knowledge through interactive programming projects.

  • Event listeners
  • User input
  • Arrays and objects
  • Conditional logic

Example student projects include:

  • Grade Tracker Web Application
  • Login Screen UI
  • Quiz Applications
  • Interactive dashboards

Phase 3: Industry Simulation Projects

Students complete larger applications that simulate real software development environments.

  • Kanban Task Manager
  • JavaScript Quiz Game
  • Data tracking applications
  • Game scoreboards

Phase 4: Certification Readiness

Students complete certification-style assessments and debugging exercises that mirror the structure of the industry exam.

  • Timed coding challenges
  • Practice certification tests
  • Code tracing exercises
  • Peer code reviews

Student Portfolio Evidence

Students demonstrate readiness through completed projects such as:

  • Grade Tracker Application
  • Kanban Task Manager
  • JavaScript Quiz Game
